# Инициализация слайдера
При загрузке страницы в слайдер необходимо отрисовать

1. Первое изображение из категории
2. Navigation bullet слайдера (первая должна быть активна);
3. Счетчик слайдов (0/100);

# Переключение слайдов
При клике на кнопки переключения слайдов необходимо изменять:

1. Изображение
2. Активную navigation bullet
3. Счетчик слайдов

Слайдер должен быть зацикленным - если изображения закончились нужно начинать показ с начала

# Navigation bullet

При клике на navigation bullet необходимо сделать ее активной и поменять изображение и счетчик на соответсвующее номеру navigation bullet;


# Категории

При клике на кнопку категории должен отрисовываться слайдер для текущей категории (изображения, количество navigation bullet и счетчик слайдов)
