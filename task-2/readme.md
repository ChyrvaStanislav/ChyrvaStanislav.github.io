# Обработка скролла

1. При положении скролла страницы больше 100px добавлять к хедеру стили для тени (`class="shadow"`). При положении скролла меньше 100px тени не должно быть. 
2. При положении скролла страницы больше высоты показывать кнопку scrollToTop. При меньше - скрывать.
3. При клике на кнопку - плавно скролить до начала документа.
