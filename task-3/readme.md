1. При клике на сабмит собирать все значения с инпутов в объект и выводить в консоль. Очищать все поля ввода. Страница не должна перезагружаться!
    ```js
      {
        name: 'John',
        tel: '+76546623462',
      }  
    ```
2. Реализовать событие сабмита формы при нажатии клавиши Enter [https://developer.mozilla.org/ru/docs/Web/API/HTMLFormElement/submit](https://developer.mozilla.org/ru/docs/Web/API/HTMLFormElement/submit)
3. При вводе в инпуты текста запрещать вводить спец. символы (`?!=`)
4. По расфокусу с инпута repeatPassword проверять соответствуют ли введенные данные полю password. Если нет - показывать ошибку (добавлять текст красного цвета под полем ввода). По фокусу скрывать ошибку.
